安装与配置说明

本项目基于 Go + Gin，需先完成开发环境准备。
1️⃣ 安装 Go 语言环境

官网地址： https://go.dev/dl

平台推荐命令说明
Windows 下载安装 MSI 安装包 从 官网 下载 `.msi` 安装包并执行安装，自动配置环境变量
Linux 
wget https://go.dev/dl/go1.21.0.linux-amd64.tar.gz -O go.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
 一键下载并安装 Go 到 /usr/local，写入环境变量

其他 Linux 安装方式：

  • 📦使用包管理器（适用于 Debian/Ubuntu）： 
    sudo apt update
sudo apt install golang-go

    说明：安装较为便捷，但版本可能不是最新版

  • 🧩手动安装方式： 
    wget https://go.dev/dl/go1.21.0.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.21.0.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc

    说明：自行下载解压并配置 PATH 环境变量

验证是否安装成功：

go version

输出示例：

go version go1.21.0 linux/amd64

本项目已经包含 go.mod 和 go.sum 文件，依赖包（如 Gin 和 yaml）已被声明。

方式操作命令说明
✅ 自动加载（推荐） go run main.go
或
go build		 Go 将自动识别依赖并下载
🔧 手动下载依赖(参考下面手动配置教程) go mod tidy 主动下载并清理未用依赖

你可以选择任意一种方式加载依赖。

2️⃣ 安装 Gin 框架 
go get -u github.com/gin-gonic/gin

说明：从官方仓库拉取 Gin 框架，供后续项目使用。

3️⃣ 安装 YAML 配置库 
go get gopkg.in/yaml.v3

说明：项目使用 yaml.v3 来解析配置文件 config.yaml。

4️⃣ 下载 GoBlade 项目
平台命令
Windows / Linux 
git clone https://github.com/2380494437/goblade.git (暂未上传git)

进入项目目录：

cd goblade
5️⃣ 初始化依赖 
go mod tidy

说明：整理并拉取 go.mod 中声明的依赖包。

6️⃣ 启动项目 
go run main.go

说明：启动 Web 服务，默认监听 0.0.0.0:8080，可通过浏览器访问 http://localhost:8080。

7️⃣ 调试运行（推荐使用 GoLand）
  • 使用 IDE 打开项目目录
  • 配置 main.go 为启动项
  • 添加调试断点，点击运行或调试按钮
8️⃣ 编译发布
系统命令说明
Windows 
go build -o goblade.exe main.go
 生成 Windows 可执行文件
Linux 
GOOS=linux GOARCH=amd64 go build -o goblade main.go
 交叉编译为 Linux 可执行文件

执行文件说明：

  • goblade.exe / goblade：可部署运行的主程序
  • 部署时只需保留 static、templates、config.yaml 等配置资源
✅ 完成！

你已完成安装和运行配置，可以继续开发自己的后端服务。
