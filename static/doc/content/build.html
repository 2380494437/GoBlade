编译代码（生成可执行文件）

📦 Linux → Linux 编译 
go build -o goblade main.go

生成 goblade 可执行文件，直接运行：

./goblade
💻 Windows → Windows 编译 
go build -o goblade.exe main.go

双击或在 CMD 中运行 goblade.exe

🐧 Windows → Linux 交叉编译 
SET GOOS=linux
SET GOARCH=amd64
go build -o goblade main.go

⚠️ 编译结果 不能在 Windows 执行，应传至 Linux。

📌 常见问题
  • 未设置 GOOS/GOARCH？默认编译为当前平台
  • 提示权限 denied？请用 chmod +x goblade 授权
  • 编译后报错缺包？请先 go mod tidy
