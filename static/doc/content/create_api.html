创建一个新接口

本节将指导你从零创建一个自定义 API 接口，例如：GET /api/v1/hello。
📁 第一步：创建控制器函数

在 /controllers/v1/ 目录下新建 hello.go：

// controllers/v1/hello.go

package v1

import (
    "github.com/gin-gonic/gin"
    "GoBlade/utils"
)

func HelloHandler(c *gin.Context) {
    utils.JSON(c, 200, "Hello from GoBlade", gin.H{
        "method": c.Request.Method,
        "path":   c.Request.URL.Path,
    })
}
🧩 第二步：注册路由路径

打开 /routes/v1/v1.go，添加路由绑定：

r.GET("/hello", controllers.HelloHandler)

完整路径为：/api/v1/hello

🚀 第三步：运行并访问接口
  • 运行项目：go run main.go
  • 浏览器访问：http://localhost:8080/api/v1/hello
  • 返回示例：
{
    "code": 200,
    "message": "Hello from GoBlade",
    "data": {
        "method": "GET",
        "path": "/api/v1/hello"
    }
}
📌 命名规范建议
  • 控制器函数命名：XXXHandler，如 LoginHandler
  • 文件按模块分类：如 user.go、article.go
  • 建议分版本放置：controllers/v1/、controllers/v2/
✅ 你现在已经可以创建任意接口。下一步：了解如何统一返回结构，请进入“返回 JSON 数据”页面。
