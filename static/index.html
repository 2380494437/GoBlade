GoBlade
轻量、清晰、可拓展的 Go 后端模板框架
  • 👤 作者： 菜鸟八哥
  • 🛠 技术栈： Go + Gin + MySQL + Layui
  • 🎯 项目定位： 简洁、高性能，适合中小型接口服务
  • 👥 适用人群： Go 初学者、ThinkPHP 转 Go 用户、后端开发者
  • 📂 支持功能： 多版本路由、自动建表、安装界面、错误页、静态资源托管

📈 项目优势

  • 轻量无依赖： 除 Gin 外几乎无三方依赖，适合快速部署
  • 结构清晰： 分层合理，控制器、路由、配置模块分离
  • 多版本支持： 接口支持 /api/v1, /api/v2，方便迭代升级
  • 自动建表： 无需写 SQL，首次安装自动创建表结构并补全字段
  • 易于扩展： 可无缝接入 GORM、JWT、权限系统等中间件

✅ 推荐使用场景

  • 🎯 Go 初学者练习：快速理解一个后端项目的完整结构
  • 💡 中小型项目起步：接口清晰，逻辑简洁，快速启动业务开发
  • 🚀 后台管理/系统中台：配合前端即可搭建内部业务系统
  • 📦 个人/团队脚手架：二次封装可作为你团队专属 Go 模板
